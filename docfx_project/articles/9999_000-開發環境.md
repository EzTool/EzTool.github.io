## Visual Studio Code Extension
* [Preview on Web Server][PoWS]: 提供即時的網頁檢視。
* [Bracket Pair Colorizer][BPC]: 將括號以不同顏色顯示。
* [Path Intellisense][PI]:提供相對路徑的自動建議。
* [GitLent][GL]: 提供檢視 Git 簽入歷程。
* [Prettir-Code formatter][PCf]: 提供程式統一編排格式。
